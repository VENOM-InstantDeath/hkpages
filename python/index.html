Índice de Python

¡Bienvenido!

Python es un lenguaje de programación interpretado creado por Guido Van Rossum, es multiparadigma pero mayormente orientado a objetos; es decir, en Python todo es un objeto (igual que en otros lenguajes como Ruby). Para aquellos lectores que desconozcan qué es un objeto, esto se explica en la guía.

Este lenguaje es de suma importancia en la actualidad, la razón será explicada a continuación.

¿Por qué aprender Python?

Las razones para aprender Python sobran. Para empezar, es el segundo lenguaje más usado actualmente (2021) después de JavaScript, lo que a su vez significa que es el segundo lenguaje más demandado por las empresas. Esto tiene todo el sentido ya que Python es un lenguaje que cuenta con una sintaxis sumamente sencilla de leer y aprender, lo que significa que es una excelente forma de comenzar en la programación. Por supuesto, que sea bueno para comenzar no lo hace un lenguaje poco serio. Es por su nivel de abstracción que Python es un lenguaje fácil de leer, pero a parte esto trae otra ventaja que no sólo es aprovechada por los principiantes sino por los programadores que trabajan con manipulación de datos, data science y otras cosas. Esta ventaja es que al ser un lenguaje tan abstracto, el programador puede enfocarse más en qué quiere hacer en vez de cómo lo quiere hacer. Esto es un punto importante, lo que puede hacerse en Python con 2 líneas de código en otros lenguajes se hace con 10.

Python es un lenguaje multipropósito, se usa para desarrollar aplicaciones de escritorio, manipulación de datos, redes neuronales, inteligencia artificial e incluso se lo puede usar en backend para desarrollo web. Hasta ahí hay varias ventajas

  1. Es el segundo lenguaje más demandado por las empresas
  2. Sintaxis sencilla de leer y entender
  3. Alto nivel de abstracción
  4. Multipropósito

¿Dónde aprender Python?

Hay muchas guías y muchas fuentes desde las cuales se puede aprender Python, pero esta en particular es una guía muy completa y fácil de entender que guiarán hasta al usuario más novato, incluso a quienes tienen 0 nociones de programación. Al terminar de leer la guía podrán afirmar con seguridad que saben Python
